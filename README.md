# Human-Centered AI To-Do App

## Overview

A simple AI-powered productivity app designed for students to manage tasks and improve productivity.

## Problem Statement

Students often struggle with organizing daily tasks and maintaining a productive study routine.

## Solution

The application provides task management, AI suggestions, and progress tracking in a simple and user-friendly interface.

## Features

* Task Management
* AI Suggestions
* Progress Tracking
* User-Friendly Dashboard
